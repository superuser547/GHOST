# GHOST — конструктор отчётов по ГОСТ/МИСИС

GHOST — локальное web-приложение, которое помогает собирать учебные отчёты по ГОСТ и требованиям НИТУ МИСИС из готовых блоков.
Пользователь формирует структуру документа, настраивает параметры и получает корректно оформленные документы в `.docx` и `.pdf`.
Проект полностью работает офлайн и опирается только на open source-инструменты.

## Требования и спецификация

Полное функциональное описание и техническое задание см. в файле [REQUIREMENTS.md](./REQUIREMENTS.md).
Этот документ является единственным источником правды по требованиям и архитектуре проекта.

## Обзор архитектуры

- `backend/` — серверная часть на Python/FastAPI, отвечает за модели отчёта, валидацию, генерацию Markdown,
  вызов Pandoc для получения DOCX/PDF и REST API для фронтенда.
- `frontend/` — SPA на React/TypeScript: UI-конструктор отчёта, панель настроек, блоков,
  а также отображение ошибок/предупреждений и взаимодействие с backend через API.

Сейчас эти директории ещё логически пустые, реализация будет добавляться в следующих коммитах.

## Статус

Проект находится на ранней стадии разработки. На данный момент настроен только базовый каркас репозитория.
Реализация backend и frontend будет добавляться поэтапно в отдельных ветках и коммитах.

## Структура

- `backend/` — будущий код серверной части (FastAPI, генерация отчётов).
- `frontend/` — будущий код клиентской части (React + TypeScript).
- `docs/` — методические материалы (презентации, образцы отчётов, шаблоны титульных листов и т.д.).
- `REQUIREMENTS.md` — полный список требований (ФТ + ТЗ) к проекту.
